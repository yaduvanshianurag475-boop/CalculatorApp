# Calculator App

A simple console-based Calculator built with Java.

## Features
- Addition
- Subtraction
- Multiplication
- Division
- Division by zero error handling
- Calculate again option

## How to Run
1. Clone the repository
   git clone https://github.com/yaduvanshianurag475-boop/CalculatorApp.git
2. Compile
   javac src/Main.java
3. Run
   java -cp src Main

## Technologies Used
- Java
- Scanner class
- Switch statements
